# Happy Birthday, Ray!
## A game following Henry, a guy getting presents for his friend Ray.
Get two gifts for your friend by exploring the map and interacting with objects by pressing [space]. 
<br>
Play here at: https://apple-person.itch.io/happy-bday-ray
